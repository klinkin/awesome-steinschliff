# Документация

Этот репозиторий использует `docs/` как единый центр документации.
Файлы в корне (`DEVELOPING.md`, `CONTRIBUTING.md`, `COMMANDS_USAGE.md`) оставлены как короткие входные страницы.

## Разделы

- **Разработка**: [development.md](development.md)
- **Контент (структуры, YAML, snow conditions)**: [content.md](content.md)
- **CLI и команды**: [commands.md](commands.md)
- **Как внести вклад (Issues/PR)**: [contributing.md](contributing.md)

## Быстрые команды

См. полный список: `just help`.

- **Установка зависимостей**: `just bootstrap`
- **Быстрый прогон проверок**: `just check`
- **Линтинг/типы/формат**: `just lint`, `just format`
- **Тесты**: `just test`
- **Сборка README**: `just build`
